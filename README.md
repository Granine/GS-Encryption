# GS-Encryption
A Good and Simple Encryption algorism that encrypt and decrypts any file or data based on a user defined or randomly generated password
# WIP
## Usage Direction
### Powershell
To run encrypter:
`
python3 gs_encrypt.py [password] [target file] [*options]
`
